下面这份只设计 **Image Space**，暂时不涉及后续 (PP/FF/PF/FP) 四路与 SLiM 的融合。我把 HIMO、CleanDIFT、MIFNet 的原始思想先拆开，再给出一个可以直接实现的 **Invariant Image Space v1**。

# Invariant Image Space 完整设计稿

## 1. 目标定义

Image Space 的目标不是再训练一个普通图像 backbone，而是学习一个与宿主 matcher 解耦的稠密表征：

[
P=E_{\mathrm{img}}(I),
]

使其同时具有：

[
\boxed{
\text{物理结构稳定性}
+
\text{几何不变性}
+
\text{基础模型语义先验}
}
]

最终对输入：

[
I\in\mathbb{R}^{B\times1\times H\times W}
]

输出：

[
P\in\mathbb{R}^{B\times128\times H/8\times W/8},
]

同时输出两个辅助量：

[
R\in[0,1]^{B\times1\times H/8\times W/8},
]

[
O\in\mathbb{R}^{B\times2\times H/8\times W/8}.
]

分别表示：

* (P)：最终模态不变 Image Space descriptor；
* (R)：结构可重复性 Repeatability；
* (O)：局部主方向表示。

对于当前 SLiM 的 512×512 输入：

```text
Image      [B,1,512,512]
                  ↓
Image Space
                  ↓
P          [B,128,64,64]
R          [B,  1,64,64]
O          [B,  2,64,64]
```

选择 (1/8) 是为了后续与 SLiM 当前 coarse feature 对齐；当前 SLiM 的 coarse stage 本身也是第三级、192维的 (1/8) 特征。

# 2. Image Space 的理论来源必须先分清

整个 Image Space 的思想来自三个不同方向，但不能简单相加。

## 2.1 HIMO提供：低层物理不变性的结构假设

HIMO的核心观察不是“Log-Gabor很好用”，而是：

> 跨模态图像的亮度、纹理、梯度符号甚至局部形态可能发生很大变化，但大量局部结构的**主方向关系**仍具有较高稳定性。

因此HIMO把方法核心放在 Intrinsic Major Orientation，之后再围绕 IMO 构建 DoFS、PolarP 和 CDMS。

HIMO完整链条实际上是：

```text
Image
  ↓
IMO
Intrinsic Major Orientation
  ↓
DoFS
Difference-of-Feature Suppression
  ↓
Keypoint Enhancement
  ↓
PolarP Descriptor
  ↓
CDMS Multi-scale Matching
```



我们的 Image Space **只继承前三个核心思想**：

[
\text{主方向}
+
\text{奇偶结构}
+
\text{跨尺度稳定性},
]

不复制 PolarP 和 CDMS。

原因是 PolarP 是围绕稀疏关键点构造的局部极坐标描述子，而我们需要：

[
H/8\times W/8
]

的 detector-free 稠密特征。

---

## 2.2 CleanDIFT提供：干净的高层语义知识

标准 DIFT 类扩散特征通常先把干净图像 (x_0) 加噪：

[
x_t=
\sqrt{\bar\alpha_t}x_0+
\sqrt{1-\bar\alpha_t}\epsilon,
]

然后从扩散 U-Net 中间层提取特征。

CleanDIFT指出，加噪不仅损失输入信息，还使特征依赖 timestep。它将原扩散模型复制为一个可训练的 clean feature extractor，让学生接收干净图像，而冻结扩散模型接收带噪图像，通过 timestep-conditioned projection head 对齐多个时间步的内部表示。([CVF开放获取][1])

其核心训练目标可以概括为：

[
\mathcal L_{\mathrm{CleanDIFT}}
===============================

*

\sum_{k=1}^{K}
\operatorname{sim}
\left[
\operatorname{Proj}^{(k)}
\left(
F_c^{(k)}(x_0),t
\right),
F_t^{(k)}(x_t,t)
\right].
]

训练后通常去掉这些 timestep projection heads，直接从干净输入中提取无噪声、无需选择 timestep 的内部特征。([CVF开放获取][1])

但这里必须纠正一个容易混淆的点：

> **CleanDIFT本身不是专门提出的“跨任意模态不变特征模型”。**

它证明得更直接的是：

[
\text{clean diffusion feature}
\rightarrow
\text{stronger semantic correspondence representation}.
]

因此我们应该把 CleanDIFT 定位成：

[
\boxed{\text{Semantic Foundation Teacher}}
]

而不是直接声称：

[
\boxed{\text{Cross-modal GT Teacher}}
]

---

## 2.3 MIFNet提供：单模态训练利用扩散先验的可行性

MIFNet真正与我们研究问题高度相关。

它使用：

```text
Frozen keypoint detector
+
Stable Diffusion v2 latent features
```

然后通过 LFA 和 CHA 学习 modality-invariant descriptor，并仅使用单模态图像及随机变换生成训练对。

MIFNet发现直接融合 diffusion feature 和 base feature 效果并不好，因为扩散特征虽然具有较强语义信息，但比较粗糙、有噪声，而且缺少像素级匹配所需的局部几何细节。它用 GMM-based LFA 改善扩散特征分布，再通过多层 self/cross attention 的 CHA 与 base feature 交互。

其初始融合大致是：

[
F^m=
\left[
PE(P_{\mathrm{loc}})+F^r
\right]
+
\left[
PE(P_{\mathrm{loc}})+MLP(F^b)
\right],
]

然后通过多层 attention 逐渐更新 (F^m)。

MIFNet的另一个重要结论反而指出了我们的机会：

> 当跨模态外观差异导致 keypoint repeatability 较差时，仅增强 descriptor 仍然受到限制。

所以我们的 Image Space 不应该复制 MIFNet。

我们要解决的是：

[
\boxed{
\text{Dense physical repeatability}
+
\text{semantic invariance}
}
]

而不仅是：

[
\text{Keypoint descriptor enhancement}.
]

# 4. 第一部分：Physics-Anchored Orientation Encoder

这是 Image Space 的核心。

---

## 4.1 HIMO原始物理逻辑

HIMO首先使用 Co-occurrence Filter 保留主要结构、抑制不稳定细节。

其基本形式可概括为一个结构保持的加权滤波：

[
I_p^{co}
========

\frac{
\sum_{q\in W_p}
G_{\sigma_s}(p,q)
M(I_p,I_q)
I_q
}{
\sum_{q\in W_p}
G_{\sigma_s}(p,q)
M(I_p,I_q)
},
]

其中 (M(I_p,I_q)) 根据灰度共现统计调节像素之间的关联。然后HIMO在深、浅表征上利用多尺度 Log-Gabor 的奇偶响应提取 edge 和 contour 两种互补结构。

我们不应该简单：

```text
CoF
→ Log-Gabor
→ 输出HIMO图
```

因为那仍然是传统算法。

---

## 4.2 我们的改造：Physics-Anchored Learnable Quadrature Bank

采用：

[
K_{s,k}^{e/o}
=============

K_{s,k,\mathrm{LG}}^{e/o}
+
\Delta K_{s,k}^{e/o}.
]

其中：

* (K_{\mathrm{LG}})：Log-Gabor-inspired 初始化；
* (\Delta K)：可学习残差。

更进一步，为了保证不同方向之间有明确结构关系，不让八个方向卷积核独立乱学：

[
K_{s,k}^{e/o}
=============

\mathcal R_{\theta_k}
\left(
K_{s,0}^{e/o}
\right),
]

其中：

[
\theta_k=\frac{k\pi}{K},
\qquad K=8.
]

即只学习 canonical kernel，再旋转生成不同方向。

这意味着网络初始状态具有：

[
\text{HIMO/Log-Gabor方向先验},
]

训练后允许根据实际匹配任务调整。

推荐三个尺度：

[
s\in{1,2,4}.
]

得到：

[
E_{s,k}=I_s*K^e_{s,k},
]

[
O_{s,k}=I_s*K^o_{s,k}.
]

联合能量：

[
A_{s,k}
=======

\sqrt{
E_{s,k}^{2}
+
O_{s,k}^{2}
+
\epsilon
}.
]

这里：

* Odd response 更偏向 edge；
* Even response 更偏向 line/contour；
* (A) 对响应符号不敏感。

这就是 HIMO 奇偶结构思想的神经化版本，而不是固定规则组合。HIMO原本正是利用 odd/even 特性同时覆盖跨模态中的 edge 与 contour 变化。

# 5. Neural Odd-Even Coupling

HIMO最终对 odd/even orientation 进行规则式耦合。

这里改成可学习耦合。

对于每个尺度和方向：

[
U_{s,k}
=======

\phi
\left(
|E_{s,k}|,
|O_{s,k}|,
A_{s,k}
\right).
]

然后预测：

[
[g^e_{s,k},g^o_{s,k}]
=====================

\operatorname{Softmax}
\left(
G(E_{s,k},O_{s,k})
\right).
]

最终：

[
C_{s,k}
=======

g^e_{s,k}\odot\phi_e(E_{s,k})
+
g^o_{s,k}\odot\phi_o(O_{s,k}).
]

含义是：

> 当前位置更应该相信 edge response 还是 contour response，由网络判断。

但网络不能彻底放弃物理结构，因此输入仍然来自强约束 quadrature bank。

# 6. Soft Intrinsic Major Orientation

对于：

[
C_s=
{C_{s,1},...,C_{s,K}},
]

计算方向概率：

[
\pi_{s,k}(x)
============

\frac{
\exp(C_{s,k}(x)/\tau_o)
}{
\sum_j
\exp(C_{s,j}(x)/\tau_o)
}.
]

不能使用硬：

[
\arg\max_k,
]

因为需要端到端训练。

主方向采用 doubled-angle 表达：

[
v_s(x)
======

\begin{bmatrix}
\sum_k\pi_{s,k}\cos2\theta_k\
\sum_k\pi_{s,k}\sin2\theta_k
\end{bmatrix}.
]

于是：

[
\hat\theta_s
============

\frac12
\operatorname{atan2}
(v_y,v_x).
]

这样天然满足：

[
\theta\equiv\theta+\pi.
]

这对应 HIMO 中一个非常关键的问题：跨模态中梯度可能发生方向反转，但所在直线方向仍然保持一致。HIMO正是观察到这种 orientation 的稳定性，并将其作为核心低层不变量。

---

# 7. Neural Validness

HIMO中 stable/prominent orientation 很重要。

我们的 Image Space 可以从方向分布本身定义一个可微结构置信度：

[
V_s(x)
======

\left|
\sum_k
\pi_{s,k}(x)
\begin{bmatrix}
\cos2\theta_k\
\sin2\theta_k
\end{bmatrix}
\right|_2.
]

如果所有方向差不多：

[
V\rightarrow0.
]

如果存在明显主方向：

[
V\rightarrow1.
]

这可以视为我们自己的：

> **Neural Orientation Validness**

它不等于直接复制 HIMO 原始 Validness，而是把“主方向是否明确”变成连续、可学习的物理置信度。

---

# 8. Orientation-Axis Canonicalization

这是我认为 Image Space 中最值得成为核心创新的模块。

HIMO的 PolarP 会围绕关键点建立主方向，然后旋转 descriptor 的空间结构和方向划分，实现 rotation adaptation。

但 detector-free 网络无法为4096个 coarse token逐个重新裁剪 PolarP patch。

因此我们做：

# Dense Orientation-Axis Canonicalization

假设：

[
C_s(x)
======

[C_{s,0},...,C_{s,K-1}].
]

图像发生旋转后，方向响应的主要变化可以表现为 orientation channels 的循环平移。

定义：

[
\delta_s(x)
===========

\frac{K\hat\theta_s(x)}{\pi}.
]

然后进行连续可微 circular shift：

[
\widetilde C_{s,k}(x)
=====================

\operatorname{CircularInterp}
\left(
C_s(x),
k+\delta_s(x)
\right).
]

不是：

```python
torch.roll()
```

而是在：

[
\lfloor\delta\rfloor,
\qquad
\lceil\delta\rceil
]

两个循环位置之间线性插值。

最终：

```text
旋转前 orientation response
      ↓
旋转后 channel发生循环位移
      ↓
根据soft IMO反向对齐
      ↓
Canonical orientation representation
```

这样我们把 HIMO：

[
\text{keypoint-level rotational adaptation}
]

改造成：

[
\boxed{
\text{dense differentiable orientation canonicalization}
}
]

这是 Image Space 最核心的原创设计之一。

# 9. Cross-Scale Stability：神经化 DoFS

HIMO的 DoFS 不是简单“多尺度特征”。

它使用构建 IMO 时不同尺度的中间特征进行 inter-layer differencing，从而抑制尺度不稳定的位置、增强稳定关键点。

我们的 detector-free 版本不再输出 keypoint weight，而输出稠密稳定性。

得到三个 canonicalized feature：

[
Z_1,Z_2,Z_3.
]

全部变成：

[
[B,C_s,H/8,W/8].
]

先归一化：

[
\bar Z_s=
\frac{Z_s}{|Z_s|_2}.
]

计算：

[
a_{12}
======

\bar Z_1^\top\bar Z_2,
]

[
a_{23}
======

\bar Z_2^\top\bar Z_3,
]

[
a_{13}
======

\bar Z_1^\top\bar Z_3.
]

定义一个显式的尺度稳定度：

[
S_{\mathrm{scale}}
==================

\frac{
a_{12}+a_{23}+a_{13}
}{3}.
]

也可以定义：

[
D_{\mathrm{scale}}
==================

1-S_{\mathrm{scale}}.
]

这就是我们的：

> **Differentiable Dense DoFS analogue**

然后学习尺度权重：

[
[w_1,w_2,w_3]
=============

\operatorname{Softmax}
\left(
G_{\mathrm{scale}}
[
Z_1,Z_2,Z_3,
S_{\mathrm{scale}}
]
\right).
]

最终：

[
P_{\mathrm{phy}}
================

\sum_{s=1}^{3}
w_sZ_s.
]

因此不同位置可以自动选择不同有效尺度。

# 13. Semantic Relation Distillation

原因是 diffusion feature 和 dense matching descriptor 的职责不同。

MIFNet也明确发现 diffusion feature 有语义不变性，但缺少精细局部几何，因此需要与 base descriptor 进一步融合。

我们只蒸馏关系。

将 CleanDIFT teacher 特征 resize/pool 成：

[
D_i\in\mathbb{R}^{256\times C_D},
]

即16×16。

Semantic Student：

[
S_i^{16}
========

Pool(S_i)
\in
\mathbb{R}^{256\times C_S}.
]

分别计算：

[
Q_T^{01}
========

\operatorname{Softmax}
\left(
\frac{
\bar D_0\bar D_1^\top
}{
\tau_T
}
\right),
]

[
Q_S^{01}
========

\operatorname{Softmax}
\left(
\frac{
\bar S_0\bar S_1^\top
}{
\tau_S
}
\right).
]

Semantic distillation：

[
\mathcal L_{\mathrm{sem}}
=========================

KL
\left(
\operatorname{sg}(Q_T^{01})
\Vert
Q_S^{01}
\right)
+
KL
\left(
\operatorname{sg}(Q_T^{10})
\Vert
Q_S^{10}
\right).
]

这里学习的是：

> CleanDIFT认为哪些区域在高层语义关系上对应。

而不是复制某个具体扩散 feature vector。

# 14. 为什么使用 CleanDIFT 而不是原始 DIFT

这一选择需要在论文里明确。

原始 diffusion feature 的核心问题是：

[
x_0
\rightarrow
x_t
\rightarrow
\text{feature},
]

所以：

* 随机噪声进入 feature；
* timestep 是超参数；
* 不同任务最优 timestep 不一样。

CleanDIFT把多个 timestep 的信息压缩到一个 clean-input feature extractor 中，因此更适合作为稳定的 semantic teacher。([CVF开放获取][1])

但我们的创新不在 CleanDIFT 本身。

我们的变化是：

[
\boxed{
\text{CleanDIFT feature}
\rightarrow
\text{relation teacher}
\rightarrow
\text{lightweight image-space student}
}
]

而非：

[
\text{CleanDIFT feature}
\rightarrow
\text{直接加入matcher}.
]

# 15. Physical-Semantic Fusion

得到：

[
P_{\mathrm{phy}}
]

和：

[
S_{\mathrm{sem}}.
]

不直接 concat。

首先把 semantic feature 压缩成较粗 token：

[
S^{16}\in\mathbb{R}^{256\times C}.
]

Physical tokens：

[
P_{\mathrm{phy}}
\in
\mathbb{R}^{4096\times C}.
]

做一次：

# Physical-Queried Semantic Aggregation

[
A_{\mathrm{sem}}
================

\operatorname{MHA}
\left(
Q=P_{\mathrm{phy}},
K=S^{16},
V=S^{16}
\right).
]

复杂度是：

[
4096\times256,
]

而不是4096×4096。

然后预测 gate：

[
G_s
===

\sigma
\left(
H_s[
P_{\mathrm{phy}},
A_{\mathrm{sem}},
R
]
\right).
]

最终：

[
P
=

\operatorname{LN}
\left[
W_pP_{\mathrm{phy}}
+
\alpha_s
G_s\odot
A_{\mathrm{sem}}
\right].
]

最后 L2 normalize：

[
\bar P(x)
=========

\frac{P(x)}
{|P(x)|_2+\epsilon}.
]

推荐：

[
\alpha_s
]

初始化为0.1左右，而不是完全0。

因为 Semantic Student 本身已经有独立蒸馏监督，不需要担心初始语义分支完全随机破坏物理特征。

```text
│                                    │

         ↑                            │

                                      ↓

                                      ↓

```

# 17. Image Space损失

完整训练损失建议：

[
\boxed{
\mathcal L_{\mathrm{IMG}}
=========================

\mathcal L_{PP}
+
\lambda_o\mathcal L_{\mathrm{ori}}
+
\lambda_r\mathcal L_{\mathrm{rep}}
+
\lambda_s\mathcal L_{\mathrm{sem}}
}
]

第一版只需要四项。

---

## 17.1 Dense Physical Matching Loss

对于：

[
P_0,P_1,
]

计算：

[
S_{PP}
======

\frac{
\bar P_0
\bar P_1^\top
}{\tau}.
]

直接使用当前 SLiM 的 coarse GT 和 matching loss。

当前 SLiM 本身就是通过 coarse similarity matrix 与已知 GT 对应关系进行双方向 partial softmax focal supervision。

因此：

[
\mathcal L_{PP}
===============

\mathcal L_{\mathrm{coarse}}
(S_{PP},M_{GT}).
]

它保证：

> Image Space 最终不是一个纯语义表示，而是真正能够做精确 correspondence。

---

## 17.2 Orientation Equivariance Loss

已知：

[
x_1=H(x_0).
]

局部 Jacobian：

[
J_H(x)
======

\frac{\partial H(x)}
{\partial x}.
]

图0预测方向：

[
u_0=
[\cos\theta_0,\sin\theta_0]^\top.
]

经过 Homography：

[
u_1^*
=====

\frac{
J_Hu_0
}{
|J_Hu_0|
}.
]

因为方向具有：

[
u\equiv-u,
]

所以：

[
\mathcal L_{\mathrm{ori}}
=========================

1-
\left|
u_1^\top u_1^*
\right|.
]

这让网络明确学习：

[
\text{orientation equivariance}
\rightarrow
\text{canonicalization invariance}.
]

---

## 17.3 Repeatability Loss

第一项是几何一致：

[
\mathcal L_{\mathrm{rep-cons}}
==============================

\left|
R_0-
W(R_1,H^{-1})
\right|_1.
]

但这会有：

[
R\equiv1
]

的塌缩解。

因此从匹配分布产生 soft target。

对于位置 (i) 的 GT：

[
j^*,
]

定义：

[
m_i
===

## S_{PP}(i,j^*)

\operatorname{LSE}*{j\notin\mathcal N(j^*)}
S*{PP}(i,j).
]

然后：

[
R_i^*
=====

\sigma
\left(
\frac{m_i}{\tau_r}
\right).
]

使用 stop-gradient：

[
\mathcal L_{\mathrm{rep-rank}}
==============================

BCE
\left(
R,
\operatorname{sg}(R^*)
\right).
]

最终：

[
\mathcal L_{\mathrm{rep}}
=========================

\mathcal L_{\mathrm{rep-cons}}
+
\beta
\mathcal L_{\mathrm{rep-rank}}.
]

这样 Repeatability 学的是：

[
\boxed{
\text{稳定}
+
\text{可重复}
+
\text{有判别力}
}
]

而不是简单边缘强度。

# 19. 与 HIMO 的本质区别

| HIMO                    | Image Space                                |
| ----------------------- | ------------------------------------------ |
| 手工 CoF                  | 可学习结构编码                                    |
| 固定 Log-Gabor            | Physics-anchored learnable quadrature bank |
| 规则 IMO                  | Soft learnable IMO                         |
| Sparse main orientation | Dense orientation field                    |
| PolarP rotation         | Orientation-axis canonicalization          |
| DoFS keypoint weighting | Dense cross-scale stability                |
| Sparse descriptor       | 128-d dense descriptor map                 |
| 无语义信息                   | CleanDIFT semantic relation distillation   |

所以不能写成：

> We neuralize HIMO.

更准确的是：

> **We reformulate the physical principles underlying HIMO into a differentiable dense invariant representation.**

# 20. 与 MIFNet 的本质区别

MIFNet：

```text
Base Keypoint Descriptor
+
Stable Diffusion Feature
↓
GMM LFA
↓
CHA attention fusion
↓
Final Keypoint Descriptor
```



我们的 Image Space：

```text
Physical Inductive Architecture
              +
CleanDIFT Training-only Relation Teacher
              ↓
Dense Invariant Image Space
              ↓
P + Repeatability
```

核心区别：

1. 不依赖已检测关键点；
2. 显式解决 repeatability；
3. 不直接融合 diffusion feature；
4. 不复制 diffusion feature；
5. CleanDIFT 默认只作为训练教师；
6. 输出一个独立的 dense invariant space；
7. 后续还会通过 (PF/FP) 将不变性迁移到宿主 matcher。

MIFNet自己也指出，在外观差异导致关键点重复性不足时，其以 descriptor 增强为主的机制会受到限制。

---

# 21. 与 CleanDIFT 的本质区别

CleanDIFT解决：

[
\text{How to extract better diffusion features?}
]

我们解决：

[
\text{How to transfer foundation semantic knowledge into a physically constrained dense matching representation?}
]

所以 CleanDIFT只是：

[
T_{\mathrm{semantic}}.
]

最终推理：

[
\boxed{
I
\rightarrow
E_{\mathrm{img}}
\rightarrow
P,R,O
}
]

不需要执行 diffusion model。

不过这个结论必须实验验证。我建议保留一个重要上界消融：

```text
Teacher-only distillation
vs
Inference-time CleanDIFT
```

因为 MIFNet在推理时直接利用 foundation feature，而我们如果完全蒸馏到轻量学生，可能损失一部分真正的跨域知识。这是目前 Image Space 设计中最大的技术风险，不应提前假定 distillation 一定完整保留 CleanDIFT 能力。

## 最终一句话定义

> **Invariant Image Space通过将HIMO中的主方向、奇偶结构和多尺度稳定性重新表述为可微稠密物理表征，并利用CleanDIFT作为训练期语义关系教师，将扩散模型的高层世界知识蒸馏至轻量语义学生，最终形成兼具局部物理稳定性、旋转尺度鲁棒性和区域语义判别性的模态不变稠密图像空间。**

我认为这版已经足够作为 Image Space 的正式设计基线。真正实施前，最需要进一步推敲的只有一个模块：**Physics-Anchored Quadrature Bank 到底采用“固定 Log-Gabor + learnable residual”，还是完全可学习 steerable quadrature filters**。前者更稳，后者创新性更高，但训练风险也明显更大。
